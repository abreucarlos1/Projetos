Introdução

O que são Stored Procedures ?

Uma Stored Procedure, é um conjunto de comandos SQL, que realizam uma determinada operação, ou seja é um programa em SQL compilado, que pode ser executado manualmente ou por outros programas.

Quais as vantagens de usar Stored Procedures ?

  • Uma Stored Procedure pode ser utilizada por vários programas.
  • Uma alteração atualiza automaticamente todos os programas que usam essa Stored Procedure.
  • Melhora o desempenho, porque residem no servidor de banco de dados.
  • Uma vez executada no servidor, os próximos acessos se tornam mais rápidos, pois o plano de execução fica na memória.
  • Substituem os scripts complexos do ASP, usados para manipular o banco de dados.

O que é necessário para executar Stored Procedures ?

  • Um banco de dados (Nos exemplos foi utilizado o Microsoft® SQL Server 7.0™)
  • Uma conexão ODBC
  • Instalar o Personal Web Server no Windows 98 ou o IIS no Windows NT
